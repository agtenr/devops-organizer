## Observation
O5 ("dependencies named or explicitly 'none'") was the sole reason a well-formed story (all other objective criteria pass, all judgment criteria Good, readiness 90) was declined. The story was silent on dependencies rather than declaring "none", which O5 counts as a miss. This is a recurring shape: human authors rarely ritually write an explicit "no dependencies" line, so otherwise-complete stories decline on O5 alone and bounce back only to add one sentence. The signal is that O5's "silence is a miss" is easy to trip on high-quality stories — worth watching whether the friction is intended (forces explicit dependency thinking) or an authoring-format gap that a story template default would close.
