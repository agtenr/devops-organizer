## Observation
Objective criterion O5 ("dependencies named or explicitly none") and the hard dependency gate (step 4) can diverge. This story satisfied O5 by naming its dependency on #30 in the description prose, but the dependency gate saw DEPS_VERDICT: NONE because #30 was attached as a "Related" link, not a "Predecessor" link. So a story can legitimately pass O5 while the automated gate never evaluates the dependency it declares. The cause: O5 is checked against free-text prose, whereas the gate is driven only by structured ADO Predecessor links — the two read different sources for the same concept, and authors naturally use prose + a generic Related link rather than a typed Predecessor link.
